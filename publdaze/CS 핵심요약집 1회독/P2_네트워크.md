# 🔑 네트워크 키워드 정리

## Ch1. 네트워크 계층

### 2.1.1 OSI 7계층

- 네트워크 표준 모델
- 송신부(높은 계층 -> 낮은 계층) -> 수신부(낮은 계층 -> 높은 계층)
- 데이터 캡슐화
  - 헤더(앞)
  - 트레일러(뒤)
- 계층
  - 7계층 - 응용 계층
  - 6계층 - 표현 계층
  - 5계층 - 세션 계층
  - 4계층 - 전송 계층
  - 3계층 - 네트워크 계층
  - 2계층 - 데이터 링크 계층
  - 1계층 - 물리 계층

### 2.1.2 TCP/IP 4계층

- 계층
  - 4계층 - 응용 계층
  - 3계층 - 전송 계층
  - 2계층 - 인터넷 계층
  - 1계층 - 네트워크 인터페이스 계층

## Ch2. TCP와 UDP

### 2.2.1 TCP란

| 구분           | TCP            |
| -------------- | -------------- |
| 연결 방식      | 연결형 서비스  |
| 패킷 교환 방식 | 가상 회선 방식 |
| 전송 순서      | 순서 보장      |
| 수신 여부      | 확인           |
| 통신 방식      | 1:1 통신       |
| 신뢰성         | 높음           |
| 속도           | 느림           |

### 2.2.2 핸드셰이킹

- 3-way 핸드셰이킹
  - 연결 시작
    1\. SYN(N)
    2\. ACK(N+1) + SYN(M)
    3\. ACK(M+1)
- 4-way 핸드셰이킹
  - 연결 해제
    1\. FIN
    2\. ACK
    3\. FIN
    4\. ACK

### 2.2.3 TCP 제어 방법

- 흐름 제어
  - 정지-대기
  - 슬라이딩 윈도우
- 혼잡 제어
  - AIMD
  - 느린 시작
  - 혼잡 회피
  - 빠른 회복
  - 빠른 재전송
- 오류 제어
  - 정지-대기
  - Go-Back-N ARQ
  - Selective-Repeat ARQ

### 2.2.4 UDP

| 구분           | UDP                |
| -------------- | ------------------ |
| 연결 방식      | 비연결형 서비스    |
| 패킷 교환 방식 | 데이터그램 방식    |
| 전송 순서      | 순서 바뀔 수 있음  |
| 수신 여부      | 확인하지 않음      |
| 통신 방식      | 1:1, 1:N, N:N 통신 |
| 신뢰성         | 낮음               |
| 속도           | 빠름               |

### 2.2.5 UDP의 오류 검출

- 체크섬
- 데이터 모두 더함 -> 캐리 더함 -> 1의 보수 취함 => 체크섬
