# 🔑 네트워크 키워드 정리

## Ch1. 네트워크 계층

### 2.1.1 OSI 7계층

- 네트워크 표준 모델
- 송신부(높은 계층 -> 낮은 계층) -> 수신부(낮은 계층 -> 높은 계층)
- 데이터 캡슐화
  - 헤더(앞)
  - 트레일러(뒤)
- 계층
  - 7계층 - 응용 계층
  - 6계층 - 표현 계층
  - 5계층 - 세션 계층
  - 4계층 - 전송 계층
  - 3계층 - 네트워크 계층
  - 2계층 - 데이터 링크 계층
  - 1계층 - 물리 계층

### 2.1.2 TCP/IP 4계층

- 계층
  - 4계층 - 응용 계층
  - 3계층 - 전송 계층
  - 2계층 - 인터넷 계층
  - 1계층 - 네트워크 인터페이스 계층
