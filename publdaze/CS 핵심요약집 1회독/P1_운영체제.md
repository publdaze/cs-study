### 1.1.1 운영체제란

- 운영체제
  - 소프트웨어 계층
  - 자원 관리
  - 사용자 사용 환경 제공
  - 윈도우, 맥OS, 리눅스, 유닉스

### 1.1.2 운영체제의 목적

- 처리 능력 향상
- 반환 시간 단축
- 사용 가능도 향상
- 신뢰도 향상

### 1.1.3 CPU와 메모리 구조

- CPU
  - 프로세서
- 메모리
  - 데이터 저장
  - 주 기억장치
    - 휘발성
    - 메인 메모리 (RAM)
  - 보조 기억장치
    - 비휘발성
    - SSD
    - HDD
  - 계층 구조
    - 레지스터
    - 캐시 메모리
    - RAM
    - 하드 디스크

### 1.1.4 커널과 시스템 콜

- 커널
  - OS 핵심 요소
  - 하드웨어와 프로세스 보안
  - 자원 관리
    - CPU 스케줄링
    - 메모리 관리
    - 입출력 관리
    - 파일 시스템 관리
  - 하드웨어 추상화
- 사용자 모드 ↔️ 시스템 콜 ↔️ 커널 모드
